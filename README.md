# nedo-remote-rehabilitation-server

NEDO遠隔リハビリテーションのためのサーバーサイドシステムです。

## Requirements
- Node.js v18
- Postgre SQL v14

## Usage

1. 当リポジトリをクローンしてください。
1. `npm install --save`
1. `cp dot-env-example.txt .env`
1. .envファイルの内容を適切な値に書き換えてください。
1. `npm run build`
1. `npm start`
